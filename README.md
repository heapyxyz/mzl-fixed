# mzl-fixed
`mzl-fixed` is a fixed version of [Map Zone Library](https://github.com/peace-maker/mapzonelib) which wouldn't compile on SourceMod 1.10 and above.

## Requirements
- SourceMod 1.10 or newer
- [smlib](https://github.com/bcserv/smlib)
